Calculateur De Nutri-Score

Description: Le Nutri-Score est un indicateur qui permet de evaluer la qualité d'un produit alimentaire. L'objectif de ce projet est de developper un prototype d'application web qui permet de prédire le Nutri-Score d'un produit alimentaire à partir de données entrées par l'utilisateur. Le modèle de prediction utilisé est un modèle de classification (Decision Tree) entrainé à partir de la base de donées Open Food Facts.

Contenu de l'application:

Formulaire contenant différents champs qui doivent être renseignés par l'utilisateur :

  • Energie en kj
  • Matières Grasses
  • Sucres
  • Fibres
  • Protéines
  • Sel
  • Pourcentage Fruits & Legumes

    • Une fois les champs renseignés, le Nutri-Score predit est affiché sur une autre page.
